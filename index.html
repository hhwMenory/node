读书笔记_Linux程序设计_泛读

linux-who

显示目前登入系统的用户信息

-H or –heading：显示各栏位的标题信息列
-u：显示闲置时间，若该用户在前一分钟之内有进行任何动作，则标识“.”，如
果该用户已超过24小时没有任何动作，则标志“old”
-m：同 who am i
-q or –count：只显示登入系统的帐号名称和总人数

/bin：二进制文件目录，用于存放启动系统时用到的程序
/usr/bin：用户二进制文件目录，用于存放用户使用的标准程序
/usr/local/bin：本地二进制文件目录，用于存放软件安装的程序
系统用户管理员登录后使用的 PATH 变量可能还包含存放系统管理程序的目录 /sbin 和 /usr/sbin
可选的操作系统组件和第三方应用程序可能被安装在 /opt 目录下

shell 语法

变量

  • 在 shell 使用变量之前通常并不需要声明
  • 默认情况下，所有的变量都被看作字符串并以字符串来存储
  • 在变量名之前加上一个 $ 符号来访问它的内容
  • 字符串里包含空格就必须要用引号括起来
  • 等号两边不能有空格
  • read 命令可以将用户的输入赋值给一个变量，回车键结束命令

使用 xhprof 来分析 php 性能

配置

  • extension=xhprof.so
  • xhprof.output_dir=/tmp 储存 XHProf 运行数据的默认目录

内置函数

  • xhprof_enable(XHPROF_FLAGS_CPU + XHPROF_FLAGS_MEMORY);
    • 启动 xhprof 性能分析器
  • xhprof_disable();
    • 停止 xhprof 性能采样分析器
  • xhprof_sample_enable 和 xhprof_sample_disable

yaml 语法

yaml是专门用来写配置文件的语言，非常简洁强大，远比 json 方便

基础语法规则

  • 大小写敏感
  • 使用缩进表示层级关系
  • 缩进时不允许使用 tab 键，只允许空格键
  • 缩进空格数目不重要，只要相同层级的元素左侧对齐即可
  • # 表示注释

yaml 支持的数据格式

  • 对象：一组键值对，冒号结构表示
  • 数组：一组连词线开头的行，构成一个数组，缩进可以构成子数组
  • 纯量：是最基本的、不可再分的值。字符串、布尔值、整数、浮点数、null、时间、日期

注意

  • null 用 ~ 表示
  • 时间采用 ISO8601格式，2001-12-14t21:59:43.10-05:00
  • 日期采用复合 ISO8601 格式的年、月、日表示
  • 允许使用 !! 强制转换数据类型

字符串是最常见，也是最复杂的一种数据类型，默认不用引号表示，若包含特殊字符，则需要引号

& 用来建立锚点，<< 表示合并到当前数据，* 用来引用锚点
