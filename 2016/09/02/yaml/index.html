yaml 语法

发表于 2016-09-02   |   分类于 其他

yaml是专门用来写配置文件的语言，非常简洁强大，远比 json 方便

基础语法规则

  • 大小写敏感
  • 使用缩进表示层级关系
  • 缩进时不允许使用 tab 键，只允许空格键
  • 缩进空格数目不重要，只要相同层级的元素左侧对齐即可
  • # 表示注释

yaml 支持的数据格式

  • 对象：一组键值对，冒号结构表示
  • 数组：一组连词线开头的行，构成一个数组，缩进可以构成子数组
  • 纯量：是最基本的、不可再分的值。字符串、布尔值、整数、浮点数、null、时间、日期

注意

  • null 用 ~ 表示
  • 时间采用 ISO8601格式，2001-12-14t21:59:43.10-05:00
  • 日期采用复合 ISO8601 格式的年、月、日表示
  • 允许使用 !! 强制转换数据类型

字符串是最常见，也是最复杂的一种数据类型，默认不用引号表示，若包含特殊字符，则需要引号

& 用来建立锚点，<< 表示合并到当前数据，* 用来引用锚点
