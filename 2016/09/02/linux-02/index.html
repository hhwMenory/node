Linux 学习笔记02 -- Linux 文件属性详细说明

发表于 2016-09-02   |   分类于 Linux

Linux 文件或目录的属性主要包括：文件或目录的节点、种类、权限模式、链接数量、所归属的用户和用户组、最近访问或修改的时间等内容。

inode

inode，索引节点。每个存储设备或存储设备的分区（存储设备拉硬盘、U盘等）被格式化为文
件系统后，应该有两部分：
    1. inode：用来存储数据的信息，如文件的大小、属主、读写权限等
    2. block：用来存储数据

链接文件

  • 用 ln 可以创建链接文件，语法：ln 源文件 目标文件
  • 在 Linux 文件系统中，inode 值相同的文件是硬链接文件
    • 硬链接不能为目录创建
    • 当一个其中一个硬链接文件的内容，互为硬链接文件的内容也会变化，若是删除时，其他不影响
  • -s 选项可以创建软链接，链接文件不能独立存在
